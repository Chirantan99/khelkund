Scoring Points
The players in your team will earn points as per the following rules:
Batting
Events Points
Every run scored +1 Point
Every 4 scored +2 bonus Points
Every 6 scored +4 bonus points
25 < Runs Scored < 50 +10 bonus points
50 < Runs Scored < 75 +25 bonus points
75 < Runs Scored < 100 +40 bonus points
Runs Scored > 100 +50 bonus points
S/R < 75 * -20 points
75 < S/R < 85 * -15 points
85 < S/R < 95 * -10 points
95 < S/R < 105 * -5 points
105 < S/R < 115 * 0 points
115 < S/R < 125 * +5 bonus points
125 < S/R < 135 * +10 bonus points
135 < S/R < 150 * +15 bonus points
S/R > 150 * +20 bonus points
Out On Duck -15 points
*S/R points are only applicable when the player has faced a minimum of 15 balls
Bowling
Events Points
Every Wicket +20 Points
Player takes 3 wickets +25 Bonus Points
Player takes 4 wickets +40 bonus points
Player takes 5 wickets +50 bonus points
Maiden Over +20 bonus points
Economy < 2 * +25 bonus points
2 < Economy < 3 * +20 bonus points
3 < Economy < 4 * +15 bonus points
4 < Economy < 5 * +10 bonus points
5 < Economy < 6 * +5 bonus points
6 < Economy < 7 * 0 points
7 < Economy < 8 * -5 bonus points
8 < Economy < 9 * -10 bonus points
9 < Economy < 10 * -15 bonus points
Economy > 10 * -20 bonus points
*Economy Points are only applicable when the player bowls at least 2 overs
Fielding
Events Points
1 Catch +10 Points
Run Out +10 Points
Stumping +15 Points
Man of the Match * +25 Points
